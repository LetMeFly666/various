# 费解的开关

## 问题描述
$5\times5$的$01$地图，$0$代表这个灯是关着的，$1$代表是开着的。

每次点击一盏灯，它及它的上下左右共$5$盏灯的开关状态都会发生置换。

问你能不能在$6$步之内把所有的灯点亮。

## 样例输入
```
3
00111
01011
10001
11010
11100

11101
11101
11110
11111
11111

01111
11111
11111
11111
11111
```

## 样例输出
```
3
2
-1
```

## 解题思路
不如先枚举第一行的每个灯的点与不点($2^5=32$种情况)，之后第一行就不再动了，第一行没亮的灯由第二行对应的那盏灯点亮。

最后看最后一行是否都是亮着的。

**剪枝**：中途一旦遇到步数已经大于6就退出。
